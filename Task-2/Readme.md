Student Grade Calculator
This program calculates the total marks, average percentage, and grade of a student based on marks entered for multiple subjects.

Features
1. Input marks for different subjects
2. Calculates total marks
3. Calculates average percentage
4. Displays grade based on percentage

Concepts Used
1. Loops
2. Arithmetic operations
3. If-else conditions
4. Scanner class

Technologies Used
1. Java
2. Object-Oriented Programming
3. Console-based interface
   
How to Run the Programs
1. Clone this repository
2. Open the project in any Java IDE
3. Compile and run the .java files
